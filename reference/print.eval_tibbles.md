# Printing simulation results

Prints objects created by
[`eval_tibbles()`](http://MarselScheer.github.io/simTool/reference/eval_tibbles.md)

## Usage

``` r
# S3 method for class 'eval_tibbles'
print(x, ...)
```

## Arguments

- x:

  object of class `eval_tibbles`

- ...:

  not used. only necessary to define the function consistently with
  respect to `print(x, ...)`
